# 第 2 章：安装与启动 Redis

> 本章目标：在你的电脑上把 Redis 运行起来。

---

## 一、安装前的准备

Redis 是一个**服务器程序**，安装后会在后台运行一个服务，你的程序通过连接这个服务来存取数据。

---

## 二、Windows 用户安装

Redis 官方主要支持 Linux/macOS，但 Windows 用户有以下几种方式：

### 方式 1：使用 Docker（最推荐）

如果你电脑上装了 Docker，这是最简单的方式：

```bash
docker run -d --name my-redis -p 6379:6379 redis:latest
```

**这行命令做了什么？**
- `docker run`：启动一个容器
- `-d`：后台运行
- `--name my-redis`：给容器起名叫 my-redis
- `-p 6379:6379`：把容器的 6379 端口映射到你电脑
- `redis:latest`：使用最新版的 Redis 镜像

**验证是否成功：**
```bash
docker ps
```
如果能看到 my-redis 在运行，说明安装成功。

### 方式 2：使用 WSL（Windows Subsystem for Linux）

如果你开启了 WSL，可以在 WSL 中安装：

```bash
# 更新软件列表
sudo apt-get update

# 安装 Redis
sudo apt-get install redis-server

# 启动 Redis
sudo service redis-server start
```

### 方式 3：使用第三方 Windows 版本

从 https://github.com/tporadowski/redis/releases 下载 Windows 版本。

---

## 三、macOS 用户安装

```bash
# 使用 Homebrew 安装
brew install redis

# 启动 Redis
brew services start redis
```

---

## 四、Linux 用户安装

### Ubuntu/Debian
```bash
sudo apt-get update
sudo apt-get install redis-server
sudo systemctl start redis-server
```

### CentOS/RHEL
```bash
sudo yum install redis
sudo systemctl start redis
```

---

## 五、启动 Redis

### 直接启动

```bash
redis-server
```

你会看到类似这样的输出：
```
* Ready to accept connections
```

这说明 Redis 已经启动，正在监听 **6379** 端口。

**注意：** 这样启动会占用当前终端窗口。关闭终端，Redis 也会停止。

### 后台启动

```bash
redis-server --daemonize yes
```

这样 Redis 会在后台运行，不占用终端。

---

## 六、验证 Redis 是否正常运行

打开一个新的终端，运行：

```bash
redis-cli ping
```

如果返回：
```
PONG
```

说明 Redis 正常运行！

**原理解释：**
- `redis-cli` 是 Redis 自带的命令行工具
- `ping` 是一个测试命令
- Redis 收到 `ping` 后会回复 `PONG`，就像你喊一声"在吗"，对方回"在"一样

---

## 七、停止 Redis

```bash
# 方式 1：使用 redis-cli
redis-cli shutdown

# 方式 2：如果是 Docker 启动
docker stop my-redis
```

---

## 八、常见问题

### 问题 1：端口被占用

如果启动时报错 "Address already in use"，说明 6379 端口已被占用。

**解决方法：**
```bash
# 查看谁占用了端口
netstat -ano | findstr 6379

# 或者换一个端口启动
redis-server --port 6380
```

### 问题 2：Windows 无法安装

Redis 官方不直接支持 Windows。建议使用 Docker 或 WSL。

---

## 九、本章小结

### 你需要记住的

1. Redis 默认端口是 **6379**
2. 启动命令：`redis-server`
3. 验证命令：`redis-cli ping` → 返回 `PONG` 表示正常
4. 停止命令：`redis-cli shutdown`

### 安装方式推荐

| 系统 | 推荐方式 |
|------|----------|
| Windows | Docker |
| macOS | Homebrew |
| Linux | apt/yum |
